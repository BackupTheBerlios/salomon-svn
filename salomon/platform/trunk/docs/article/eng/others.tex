\section{Related Work}

There are several commercial and non-commercial packages that
provide machine learning facilities. Some of free environments are
presented below.

Yet Another Learning Environment (\emph{YALE}), alike \emph{Weka}

% umo¿liwiaj¹ tworzenie ,,³añcuchów'' z³o¿onych z ró¿nych problemów

A rich set of operators, which allow to create ''chains'' of

% Dane, na których operuj¹ operatory s¹ dla
% nich ca³kowicie przeŸroczyste i przechowywane mog¹ byæ w
% ró¿norodnych formatach -- za odpowiednie transformacje pomiêdzy

The data layer, on which the operators work, is completely

% Dziêki operatorom i mo¿liwoœci ich ³atwego ³¹czenia przy u¿yciu
% graficznego interfejsu u¿ytkownika, a tak¿e dziêki jêzykowi

% u¿ywane do prowadzenia badañ i eksperymentów z ró¿norodnymi

% \emph{YALE} dostarcza ponad 200 operatorów, jak choæby algorytmy
% klasyfikuj¹ce, asocjacyjne, klastruj¹ce, wyszukuj¹ce regu³y czy

\emph{YALE} provides over 200 operators for classification,

% Dodatkowo mo¿liwe jest
% wykorzystanie wielu algorytmów dostarczanych przez œrodowisko

Additionally, it is possible to use many algorithms provided by

\emph{YALE} helps in finding the best set of operators for the

% Dziêki modularnej strukturze operatorów
% mo¿liwa jest wymiana pojedynczych ogniw ,,³añcuchów'' obliczeñ w celu uzyskania

Thanks to the modular structure of operators, it is possible to

% !!!!!!!!!!!! to bylo zakomentowane - ma byc? !!!!
%Du¿e mo¿liwoœci tego œrodowiska mog¹ byæ dodatkowo rozszerzane

The possibilities of this environment can extended by adding new,
user-defined plug-ins. Such plug-ins can provide new operators,

Thanks to the operators and the possibility of their linking with
the graphical user interface, thanks to the script language, which
is based on \emph{XML}, \emph{YALE} is often used in a research.

\subsection*{VINLEN}
% to system indukcyjnej bazy danych, rozwijany w
% \emph{George Mason University}~\cite{bib2}. Jest to klasyczna
% realizacja indukcyjnej bazy danych.
\emph{VINLEN} is being developed in the \emph{George Mason
University}~\cite{bib2}. It is a classical example of an inductive
database.
% Mechanizmy wnioskowania
% indukcyjnego s± zintegrowane ze standardowymi relacyjnymi
% operatorami bazodanowymi. Integracja ta opiera siê na nowych
% rodzajach operatorów zwanych operatorami generowania wiedzy
% \emph{KGO} (Knowledge Generation Operators).
The mechanisms of induction are integrated with a standard
relational database operators. The integration is based on new
kinds of operators, which are called \emph{KGO} (Knowledge
Generation Operators).
 \emph{KGO}
%  operuje na
% \emph{segmentach wiedzy} sk³adaj±cych siê z~kombinacji jednej lub
% wiêcej tabel z relacyjnej bazy danych i~wiedzy przechowywanej
% w~bazie wiedzy (por. rysunek~\ref{fig:inddb}). \emph{KGO} przyjmuje
% na wyj¶ciu jeden lub wiêcej segment wiedzy, na podstawie którego
% generuje inny segment wiedzy. Podstawowym operatorem u¿ywanym w
% systemie jest AQ21, program do indukcyjnego tworzenia regu³ na
% podstawie danych.
operates on segments of knowledge, which consist of combination of
one or more relational databases and knowledge stored in a
knowledge base (see Fig.~\ref{fig:inddb}). \emph{KGO} takes as a
input one or more segments of knowledge, which is used to generate
other knowledge segment. Rule induction algorithm \emph{AQ21} is
the main operator which is used in the system.

%????program do indukcyjnego tworzenia regu³ na
% podstawie danych.

% Indukcyjne bazy danych mog¹ byæ wspierane przez wyspecjalizowanych
% agentów (\emph{Scouts}), których zadaniem jest synteza
% i~zarz±dzanie wiedz±, która jest dostosowywana do wymagañ
% okreœlonego u¿ytkownika.
Inductive databases can be supported by specialized agents
(\emph{Scouts}), which are responsible for synthesis and
management of the knowledge, in a way that is adapted to a
specified user needs.
% System \emph{VINLEN} pozwala na tworzenie skryptów
% operuj±cych na bazach danych, wiedzy i algorytmach uczenia
% maszynowego przy u¿yciu jêzyka \emph{KGL-1} (\emph{Knowledge
% Generation Language}).
Scouts are built using scripts, which operate on data and
knowledge and can use machine learning algorithms. They are
written in a \emph{Knowledge Generation Language} (\emph{KGL-1}).

% Cech¹ charakterystyczn± tak rozwijanego systemu bêdzie mo¿liwo¶æ
% sk³adowania wiedzy w~relacyjnej bazie danych wraz z danymi,
% zadawanie zapytañ oraz manipulowanie wiedz± z~wykorzystaniem
% jêzyka \emph{KQL} lub przy u¿yciu funkcjonalnego, graficznego
% interfejsu u¿ytkownika.
What is characteristic, generated knowledge is stored in a
relational database together with the data. User have access to
them via intuitive GUI and a script language.
