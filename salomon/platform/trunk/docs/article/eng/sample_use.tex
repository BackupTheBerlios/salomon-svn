\section{Tests of the system}
%\section{Wykonane testy systemu}

System Salomon was tested using freely available data sets 
designed for testing machine learning algorithms UCI~\cite{uci}.
The plug-ins applying to decisioning tree were tested.

The task of the first plug-in was configuration of the data processing.
It enables choosing the major and minor characteristics taken into account
during building decisioning tree. This choice is made using a graphical
user interface. The second plug-in is the actual processing component -- 
it realize the ID3 algorithm. The last plug-in is responsible for
visualization of the tree created during computation.
