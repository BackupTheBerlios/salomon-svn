\section{Tests of the system}
%\section{Wykonane testy systemu}

System Salomon was tested using freely available data sets designed
for testing UCI~\cite{uci} -- machine learning algorithms.
The test concerning the decision tree plug-ins were performed.

The aim of the first plug-in was to configure the data processing. It enables choosing
both major and minor characteristics taken into account during building decision tree. The
choice is made with the use of a graphical user interface.
The second plug-in is the actual processing component
-– it provides the realization of the ID3 algorithm. The last
plug-in is responsible for visualization of the tree created during computation.
