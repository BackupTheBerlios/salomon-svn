\section{Summary}
%\section{Podsumowanie}

\emph{Salomon} system is operational, but its new versions and
plug-ins are still being developed. Thanks to the modular
structure, good flexibility is achieved. The architecture seem to
fit to the complex learning problems.

Further works will include:
\begin{itemize}
    \item developing applications to some complex tasks;
    \item implementation of the agent mechanisms;
    \item better task distribution;
    \item support for other types of knowledge representation;
    \item implementation of new machine learning algorithms;
    \item support for other types of data sources (currently only \emph{Firebird}
    RDBS is supported).
\end{itemize}
